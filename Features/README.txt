***TRAINING***
The main.m file contains the code to train the KNN,SVM, RF model for four feature extraction methods, namely HOCD, HOCD+Gabor, Gabor 
Only one of the feature extraction techniques can be run at a time.
The main.m file also provides a template in the end to test the trained model.
The error metric that is displayed at the end, gives the test error of the classifier that is used.
